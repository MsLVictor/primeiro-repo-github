Estados observados durante o exercício. 

U untracked é quando eu crio o arquivo, mas não adiciono com " git add "

M Modified é quando eu adiciono (git add <nome do arquivo>) e faço o commit (git commit -m "<comentário>") e depois disso eu edito o arquivo.

A Added indica que o arquivo novo anteriormente não rastreado, e agora foi incluido para o próximo commit.
